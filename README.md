# TaskSmith (JavaFX + SQLite)

Quick-start:
1. Install JDK 17 and Maven.
2. In project root: `mvn -q clean javafx:run`
3. Login with **demo / demo123** (auto-created).

Features:
- Login/Register with BCrypt
- Task CRUD stored in SQLite (`tasksmith.db`)
- JavaFX UI: table + form
